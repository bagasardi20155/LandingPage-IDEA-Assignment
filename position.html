Static Position

Property seperti left, top, right, dan bottom tidak akan berpengaruh.
Relative Position

Hampir sama seperti static namun Property box offset (left, top, right, dan bottom) dapat diatur.
Absolute Position

Posisinya tergantung pada parentnya dan dapat diletakkan dimana saja. Property box offset dapat digunakan.
Fixed Position

Posisinya akan tetap walau halaman di scroll
